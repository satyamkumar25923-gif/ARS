## What is this?
ARS is a student-first web tool that tracks your attendance and predicts exactly how many classes you need to attend (or can bunk) to stay safe.

## Features
- **Smart Calculator**: Tells you "Attend 5 more" or "Safe to bunk 2".
- **Risk Analysis**: Low, Medium, High, Critical Risk indicators.
- **Privacy Focused**: Data stays in your browser (LocalStorage).

## How to Run
1. Ensure you have Node.js installed.
2. Open a terminal in this folder.
3. Run:
   ```bash
   npm install
   npm run dev
   ```
4. Open the link shown (usually http://localhost:5173).

## Tech Stack
- React (Vite)
- Vanilla CSS (Premium Dark Mode)
